## :memo: Descrição
Um aplicativo para realizar autenticação de usuarios e cadastro

## :books: Funcionalidades
* <b>Funcionalidade 1</b>: Realiza o login do usuario validando antes os campos 
* <b>Funcionalidade 2</b>: Realiza o cadastro do usuario validando antes os campos 
* <b>Funcionalidade 3</b>: Realiza a saida da autenticação do usuario para a tela inicial do aplicativo

## :wrench: Tecnologias utilizadas
* React Native
* React Navigation
* Animatable
* Firebase/auth
